# 📊 Pre–Post Assessment Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing **student performance before and after an assessment** across multiple classes.

The main objective was to transform raw, unstructured score data into a **clear, structured, and insightful dashboard** that highlights performance changes and improvement patterns.

---

## 🎯 Objectives
- Compare student performance **before vs after assessments**
- Measure and visualize **improvement trends**
- Provide a **clear and interactive dashboard** for quick insights
- Ensure **data consistency** by handling missing values

---

## 🛠️ Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Excel Formulas  
- Interactive Charts  

---

## 📂 Data Preparation
To ensure accurate analysis, the following steps were performed:

- Organized raw student data into structured tables  
- Handled missing values to avoid calculation errors  
- Standardized score formats  
- Created calculated fields for:
  - Improvement Score  
  - Improvement Percentage  
  - Performance Status  

---

## 📊 Dashboard Features
The dashboard provides:

### 🔹 Performance Comparison
- Pre-test vs Post-test results
- Clear visual comparison using charts

### 🔹 Improvement Analysis
- Improvement score for each student
- Percentage-based improvement tracking

### 🔹 Class-Level Insights
- Performance trends across multiple classes
- Distribution of student performance levels

### 🔹 Visual Summaries
- Line charts for trends
- Bar charts for comparisons
- Donut charts for performance distribution

---

## 📈 Key Insights
- Easily identify students with **significant improvement**
- Detect cases of **decline or no progress**
- Compare **overall class performance trends**
- Support data-driven educational decisions
